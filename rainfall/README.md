# Rainfall

## Instructions

- Rainfall is a 42 School project which aims to further your knowledge in the world of elf-like binary exploitation
  in i386 system.
- This project includes ten mandatory and four bonus levels.
- For each level, a binary is provided you will have to exploit in order to read the `.pass` file.
- That file will provide you the next level password.

## Rainfall Virtual Machine

- To make this project, you will have to use the Rainfall VM.
- You should use an SSH connection, on port 4242.
